In diesem Post möchte ich meine Vorgehensweise bei der Lösung des Katas ["Karate Chop" von Dave Thomas][code-kata-chop] vorstellen. Im Prinzip handelt es sich um die Implementierung einer binären Suche in verschiedenen Formen. Dabei werde Python für die Realisierung verwenden. Das Ergebniss gibt es auf [GitHub][github] zur Einsicht.

# Was sind Code Katas
Da es der erste Post über Code Katas ist, möchte ich diese kurz vorstellen. In der Vergangenheit habe ich den Begriff "Code Kata" schon häufiger gehört und später  auch ausprobiert. Im Prinzip sind sie kleine Aufgaben, die mit einer beliebigen Programmiersprache gelöst werden können. Damit kann man seine Fertigkeit zum Programmieren üben und hoffentlich ausbauen. Persönlich habe ich es oft als eine kleine Herausforderung gesehen Probleme mit einer mir neuen Programmiersprache zu lösen. Damit habe ich mehr über die neue Sprache gelernt als Tutorials zu nutzen. Laut [Wikipedia][code-kata-wiki] wurde der Begriff von Dave Thomas 2007 eingeführt. Er hat auch einen [Blog][code-kata] erstellt, in dem er 21 Kata Übungen vorstellt. Ich möchte diese 21 Übungen einfach mal machen und diese dann auch hier vorstellen.
